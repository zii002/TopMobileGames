This project analyzes a dataset of top mobile games to understand downloads, ad revenue, and genre performance.
The goal is to practice SQL data analysis skills.
Dataset: TopMobileGames
It includes information about:
-Game ID
-Game Name
-Date
-Ad Revenue
-Downloads
-Genre
-Release Date

ANALYSIS STEPS
 
1️-Preview the Data
Checked the first rows of the dataset to understand structure.

2️-Data Quality Check
Checked for missing values in important columns like game name, downloads, revenue, and genre.

3️-Top 10 Games by Downloads
Found the most downloaded games using SQL aggregation.

4️-Top 10 Games by Ad Revenue
Identified games generating the highest ad revenue.

5️-Genre Analysis
Summarized downloads and revenue by game genre to see which genres are most popular.

6️-Revenue per Download
Calculated revenue per download to measure monetization efficiency.

Tools Used
-SQL server
-Kaggle
